Fundamental framework markup for a Rackspace branded apllication.

Layout Framework

The basic layout of Canon consists of 4 parts.

  1. Utility Navigation

    consists of links and tools that help the user, such as account details, log out, or help information, but are not the core sections of the application.

  2. Primary Navigation

    used for the main sections of the application or for dropdown categories that organize sections into higher level groups.

  3. Content Area

    used for your application's main data and interactions.

  4. Footer

    used for legal information, application news, and feedback links.
